# ipio

> The CLI version of [Brook GUI](https://www.txthinking.com/brook.html)

Proxy all traffic just one line command.

### Install via [nami](https://github.com/txthinking/nami)

```
nami install ipio
```

### Usage

**root/sudo/Windows in GitBash with Admin**

```
ipio --link 'brook://...'
```
